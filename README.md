# Restaurant Menu Manager

## Author
Erzhan Dokturbaev

## Project Description
Restaurant Menu Manager is a cool Java console application for managing restaurant dishes.

This program allows the user to add dishes, display the menu, update dish information, delete dishes, save data to a file, and load data from a file.

## Project Objectives
- Practice Java programming
- Use Object-Oriented Programming principles
- Implement CRUD operations
- Work with file handling
- Create a console-based management system

## Main Functions
1. Add new dish
2. Show all dishes
3. Update dish price and description
4. Delete dish
5. Save menu to file
6. Load menu from file
7. Input validation

## OOP Principles Used

### Encapsulation
Fields in Dish class are private:

- id
- name
- price
- description

Access is provided through getters and setters.

### Classes and Objects
Dish objects are created and stored in the system.

### Modularity
The project is divided into classes:

- Main.java
- Dish.java
- RestaurantManager.java

## Data Structure Used
ArrayList<Dish>

Used for dynamic storage of dishes.

## File Handling
The program uses:

menu.txt

For saving and loading data.

## Example Menu
